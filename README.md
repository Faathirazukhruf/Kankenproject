# Kanken Project - Landing Page

Ini adalah repositori untuk *landing page* **Kanken Project**, sebuah usaha sablon cetak (*screen printing shop*) yang berlokasi di Majalengka, Jawa Barat. 

## Tentang Proyek
Website ini dirancang untuk menampilkan layanan sablon cetak dari Kanken Project dengan gaya desain **Gen-Z Neobrutalism**. Desain ini menonjolkan warna-warna cerah (*neon green*, *vibrant pink*, *yellow*), garis batas (*border*) hitam yang tebal, serta bayangan asimetris yang tajam untuk memberikan kesan yang *edgy*, modern, dan *artsy*.

## Fitur Utama
- **Penjelasan Layanan**: Detail tentang sablon *custom*, proses cepat, dan bahan premium.
- **Integrasi WhatsApp**: Tombol CTA yang langsung mengarah ke nomor WhatsApp `+6282293632467` untuk kemudahan pemesanan.
- **Lokasi Workshop**: Detail alamat lengkap beserta tautan Google Maps.
- **Mobile Responsive**: Tampilan yang ramah perangkat seluler dengan CSS *Mobile-First* dan navigasi *hamburger menu* (React State).
- **SEO & AI Optimized**: Sudah dilengkapi dengan *Schema Markup* (`JSON-LD`), *Semantic HTML*, dan optimasi kata kunci lokal untuk mendominasi pencarian *Sablon Majalengka* di Google maupun platform pencarian berbasis AI (ChatGPT, Perplexity, Gemini).

## Tech Stack
- **Vite**
- **React (Komponen Reusable)**
- **Vanilla CSS (Mobile-First)**
- Font: **Space Grotesk**

## Cara Menjalankan Secara Lokal
1. Pastikan Anda telah menginstal Node.js.
2. Buka terminal dan arahkan ke direktori proyek ini.
3. Jalankan perintah instalasi dependensi:
   ```bash
   npm install
   ```
4. Jalankan *development server*:
   ```bash
   npm run dev
   ```
5. Buka `http://localhost:5173/` di *browser* Anda.

## Deployment
Proyek ini siap untuk di-*hosting* ke platform seperti **Vercel** atau **Netlify** secara langsung. Cukup hubungkan repositori GitHub ini dan Vercel akan mendeteksi *build command* Vite secara otomatis.
